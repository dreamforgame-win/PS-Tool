# UI-Link

> 一个为 Photoshop UI 生产流程打造的 CEP 插件。  
> 把图层参数、九宫格、增量导出、位置预览、AI 清晰化和自动更新，集中放进一个面板里。

## 它能做什么

UI-Link 不是单纯的“切图工具”。

它更像是 Photoshop 里的一个 UI 资源工作台，解决的是这几类高频问题：

- 图层命名不统一
- 参数分散、容易忘
- 九宫格反复调整
- 每次改一个小图也要整包重导
- 美术和客户端之间交接成本高

UI-Link 把这些动作收束成一条稳定流程：  
**配置参数 -> 预览结果 -> 扫描差异 -> 增量导出**

---

## 核心能力

### 图层参数管理

- 保存图层名称、导出类型、图集前缀、组件类型、尺寸、导出状态
- 参数写入图层元数据，不再依赖把长配置串塞进图层名称
- 支持最近名称下拉复用
- 支持位置预览与拖拽调整

### 九宫格编辑

- 读取当前选中图层
- 可视化拖拽参考线
- 支持输入精确切线数值
- 切图预览与参数联动

### 增量扫描导出

- 扫描当前 PSD 与历史导出结果的差异
- 自动区分新增、修改、不变、关闭导出的图层
- 只导出需要更新的资源，减少重复劳动

### AI 清晰化

- 接入云端 AI 模型
- 支持自定义 API、模型和提示词
- 结果可直接回写到 Photoshop 图层

### 自动更新

- 面板内检测版本
- 发现新版本后可直接拉取更新包
- 适合团队内部统一工具版本

## 为什么它更适合 UI 生产

### 更干净

图层名保持可读，不必再看到一长串参数污染图层面板。

### 更快

增量扫描只处理变化内容，不用每次整套资源重导。

### 更稳

图层参数、尺寸、位置、九宫格都能留存在元数据里，流程更可追踪。

### 更适合协作

美术配置一次，程序按统一规则接资源，减少反复沟通。

## 使用流程

### 1. 配置图层参数

在 **图层属性** 页签中：

- 选中 Photoshop 图层
- 设置名称、类型、前缀、组件类型、尺寸
- 如有需要，在位置预览中拖拽资源摆放位置
- 点击 **保存参数**

### 2. 设置九宫格

在 **九宫格** 页签中：

- 点击 **读取当前选中图层**
- 拖动参考线或输入切线数值
- 应用九宫格参数

### 3. 扫描并导出

在 **扫描导出** 页签中：

- 设置 JSON 输出目录
- 设置图片输出目录
- 点击扫描
- 选择要导出的资源
- 执行导出

### 4. 使用 AI 清晰化

在 **AI 清晰化 / AI 实验室** 中：

- 配置 API URL、API Key、模型
- 调整提示词
- 执行清晰化并回写图层

## 安装方式

### 开启 CEP 调试模式

Photoshop 需要允许加载未签名扩展。

#### Windows

在注册表中创建：

- 路径：`HKEY_CURRENT_USER\Software\Adobe\CSXS.9`
- 键名：`PlayerDebugMode`
- 值：`1`

如果你的 Photoshop 使用的 CEP 版本不同，请把 `CSXS.9` 换成对应版本。

#### macOS

执行：

```bash
defaults write com.adobe.CSXS.9 PlayerDebugMode 1
```

### 放入扩展目录

#### Windows

```text
C:\Users\<你的用户名>\AppData\Roaming\Adobe\CEP\extensions\cep-plugin
```

#### macOS

```text
~/Library/Application Support/Adobe/CEP/extensions/cep-plugin
```

### 启动插件

打开 Photoshop 后进入：

```text
窗口 -> 扩展功能 -> UI-Link
```
